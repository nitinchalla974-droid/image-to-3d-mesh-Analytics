# Single Image 3D Mesh Reconstruction

Convert a single 2D photo into a 3D human mesh using PyTorch and SAM-3D-Body.

 # Features
- Single-image 3D reconstruction
- Generates `.obj` mesh files
- Works in Google Colab or local GPU setup

# Tech Stack 
- Python
- PyTorch
- SAM-3D-Body
- NumPy
- Trimesh

## Project Structure
- notebooks -> Colab demos  
- src -> reconstruction scripts  
- data: input images  
- outputs: generated meshes
